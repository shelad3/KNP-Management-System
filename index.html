How to install

Takes less than a minute

1

Download the APK — Tap the Download button below or scan the QR code.

2

Open the file — Locate it in your Downloads folder and tap to install.

3

Allow installation — If prompted, go to Settings > Security and enable Install from unknown sources.

4

Launch & register — Open the app, create an account using your student email, and select your class.
